# Image Pixel Inspector

Image Pixel Inspectorは、画像上のピクセル座標と色情報をブラウザで確認できるシンプルなWebツールです。

画像ファイルはブラウザ内で処理され、サーバーや外部サービスへ送信されません。インストールやビルドは不要で、ダウンロード後すぐに利用できます。

## 主な機能

- 画像のドラッグ＆ドロップ、またはファイル選択による読み込み
- カーソル位置のピクセル座標とパーセント座標の表示
- カーソル位置のRGBA値と16進カラーコードの表示
- 画像サイズとズーム倍率の表示
- マウスホイールによるズーム
- マウス中ボタンのドラッグによる表示位置の移動
- 10ピクセル間隔のグリッド表示
- 読み込んだ画像と表示状態のリセット

## 導入方法

### ZIPファイルを利用する場合

1. GitHubのリポジトリページから「Code」→「Download ZIP」を選択します。
2. ダウンロードしたZIPファイルを任意の場所へ展開します。
3. 展開したフォルダー内の `index.html` をWebブラウザで開きます。

### Gitを利用する場合

```bash
git clone https://github.com/kimura-aruku/ImagePixelInspector.git
cd ImagePixelInspector
```

クローン後、`index.html` をWebブラウザで開いてください。

## 利用方法

1. 起動画面へ画像をドラッグ＆ドロップするか、「ファイルを選択」ボタンから画像を選択します。
2. 画像上へカーソルを移動すると、画面上部に座標と色情報が表示されます。
3. マウスホイールを回すと、カーソル位置を基準に画像を拡大・縮小できます。
4. マウス中ボタンを押しながらドラッグすると、画像の表示位置を移動できます。
5. 「グリッド」を有効にすると、10ピクセル間隔の赤いグリッドが表示されます。
6. 「リセット」を押すと画像を閉じ、初期画面へ戻ります。

## 動作環境

HTML5 CanvasとFile APIに対応した、一般的なデスクトップ向けWebブラウザで利用できます。

外部ライブラリ、パッケージのインストール、Webサーバーの起動は必要ありません。

## プライバシー

読み込んだ画像は利用中のブラウザ内でのみ処理されます。本ツールは画像や色情報をネットワークへ送信しません。
